**DUMA** is a modern, high-performance e-commerce platform designed for streetwear enthusiasts. It features a premium dark-themed aesthetic, smooth cinematic animations, and a seamless user experience.

## ✨ Key Features

- 🌑 **Premium Dark Theme:** Immersive streetwear-inspired UI with custom backgrounds and glassmorphism effects.
- 🎬 **Cinematic Animations:** Powered by **GSAP (GreenSock)** for high-end scroll effects and micro-interactions.
- 🛒 **Dynamic Cart System:** Real-time cart management with quantity updates and persistence.
- 📦 **Order Management:** Dedicated "My Orders" page for customers to track their purchase history.
- 🛠️ **Admin Dashboard:** Full CRUD functionality for products, category management, and sales analytics.
- 📱 **Fully Responsive:** Optimized for mobile, tablet, and desktop using Tailwind CSS.
- 💬 **Customer Support:** Integrated Customer Care section with live support CTAs and an extensive "About Us" page.
- 🗺️ **Interactive Footer:** Modern multi-column footer with social integration and quick links.

## 🚀 Tech Stack

### Frontend:

- **React 19** - Component-based architecture.
- **Tailwind CSS** - Modern utility-first styling.
- **GSAP** - Professional-grade animations.
- **React Router 7** - Seamless client-side navigation.
- **Lucide React** - Clean and consistent iconography.

### Backend (Duma-Backend):

- **Node.js & Express** - Scalable server-side logic.
- **MongoDB Atlas** - Robust cloud database for products and orders.
- **JWT Authentication** - Secure user and admin sessions.
- **Multer** - Image upload handling for product management.

## 🛠️ Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/azarrazagula/Duma-Frontend.git
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add:

   ```env
   REACT_APP_API_URL=http://your-backend-api.com
   ```

4. **Start the development server:**
   ```bash
   npm start
   ```
